# Village Defense

A **solo-developed fantasy JRPG** built with **Godot 4.5**. The game focuses on turn-based combat, a village hub, and quest-driven progression, with systems designed to be clear, modular, and easy to expand.

---

## 🎮 Overview

**Village Defense** is a single-player JRPG where the player selects a hero and defends a growing village by completing quests and battling monsters. Progression comes from new equipment, abilities, and escalating enemy challenges.

Most gameplay content is **data-driven**, defined using Godot `Resource` files instead of hard-coded logic.

---

## ✨ Features

* **Turn-based combat** with abilities, energy, and cooldowns
* **Conditional monster AI** that chooses actions based on battle state
* **Hero archetypes** with unique stats and weapons
* **Village hub** with quests, shop, and inn
* **Quest system** with structured rewards
* **Save/load system** with persistent game state

---

## 🛠️ Built With
